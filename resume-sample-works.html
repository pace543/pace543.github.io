FEC Data Application in Clojure

I have been interested in politics for a long time, and campaign finance is an issue close to my heart. When I found out that the FEC (Federal Elections Commission) released an API to pull data about campaign finance, I decided to create an application to allow people to access and manipulate that data. I decided to use Clojure because I was interested in learning a functional language. The process of learning it would be easier than a different functional language, like F#, because Clojure is a JVM language and I would still have access to familiar Java methods and classes. This project is in its beginning stages, so I am still working on making the HTTP requests to pull the data.
This project can be found in the "fec-data" repository on my GitHub account.
Link: fec-data
